 Häufige Fragen Antworten auf die wichtigsten Punkte Wie lange werden Fundstücke aufbewahrt?

Fundstücke werden in der Regel 6 Monate aufbewahrt. Danach erfolgt die weitere Verwertung.

 Welche Angaben brauche ich für eine Verlustmeldung?

Bitte geben Sie den Zeitraum, den Fundort, eine Beschreibung sowie besondere Merkmale (z. B. Farbe, Marke, Seriennummer) an.

Wie kann ich ein Fundstück abholen?

Nach erfolgreicher Zuordnung erhalten Sie eine Benachrichtigung. Vereinbaren Sie einen Termin und bringen Sie einen gültigen Ausweis mit.
